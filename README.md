# Boilerplate

Bolierplate is a project templating tool

  - Backend (Django)
  - Frontend (React)
  - Docker (development)

### Requirements
- Python 3+ (3.8 tested)

### Usage


```sh
> export PATH="<location>/Boilerplate/bin:$PATH"
> python3 boilerplate <project_name>
```

### Configuration

Edit `settings.yml` file to override default generation settings
